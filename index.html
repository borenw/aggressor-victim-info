Type & save the aggressor (noise driver) and victim (sensitive receiver) pad info for an IC or module — or one click a macro for a popular IC. Every R/C/dV takes an optional min / max (blank ⇒ use the single value). Export the CSV and load it into the Inductive-Crosstalk Spec Checker.
